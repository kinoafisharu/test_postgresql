# Строки с командами можно копировать отсюда и вставлять в окне Command Prompt в виртуальном окружении.

pip list
# 1) Выводит список установленных в системе пакетов

pip install -r requirements.txt
# 2) Устанавливает пакеты, перечисленные в файле requirements.txt

pip freeze > requirements.txt
# 3) Создает файл requirements.txt в котором перечислены пакеты, установленные в данный момент в системе
    с указанием их версий.
    Затем этот файл requirements.txt можно использовать командой 2) для установки этих же версий пакетов в другом проекте.

python.exe -m pip install --upgrade pip
# 4) Обновляет pip до последней версии.
